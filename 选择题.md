1. 数据库（DB）、数据库系统（DBS）、数据库管理系统（DBMS）之间的关系是（）。
    - A. DB包含DBS和DBMS
    - B. DBMS包含DB和DBS
    - C. DBS包含DB和DBMS
    - D. 没有任何关系

答：
```
C
```
解析：
```
数据库系统（DBS）包含数据库（DB）和数据库管理系统（DBMS）。
```
